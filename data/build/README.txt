This directory will automate data collection for a specified year.

The goal is to (for a given year) get the per-game stats of every QB, WR, RB, TE, K, and DEF.

Going to use http://www.pro-football-reference.com


How to build data for a given year:
    1. python meta.py 2005
    2. python names.py 2005
    3. python name_index.py 2005

    4. python byes.py 2005
